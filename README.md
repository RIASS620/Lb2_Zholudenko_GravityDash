Назва гри: Gravity Dash

Платформа: PC / WebGL / Mobile

Основна дія:
Гравець натискає одну кнопку, щоб інвертувати гравітацію свого персонажа і завдяки цьому уникати зіткнень із перешкодами.

Схема керування:
Single-button геймплей. Пробіл або клік миші (тап) відповідає за зміну напрямку гравітації (Up / Down).

Мета гри:
Протриматися якомога довше і набрати максимальну кількість очок, пролітаючи крізь перешкоди. Гра безкінечна.

Правила:

- Герой автоматично рухається вперед (технічно: перешкоди рухаються на нього).
- Будь-який контакт з перешкодою, стелею або підлогою призводить до кінця гри.
- Успішний проліт між блоками додає +1 очко.

Тиск — Обмеження — Наслідок:
Постійний вплив гравітації (Тиск) та обмежений простір між блоками (Обмеження) вимагають від гравця ідеального розрахунку часу. Невдача має фатальний наслідок (Програш), після якого весь прогрес скидається.

Core Loop:
Запуск → Читання розташування перешкод → Натискання для зміни гравітації → Успішний проліт (+очко) → Повторення. У разі невдачі — миттєвий рестарт.

Стиль і атмосфера:
Мінімалізм. Прості геометричні фігури. Мультяшні об'єкти на фоні неба. Спокійна, ритмічна атмосфера.

Монетизація (25¢ arcade model):
Економіка побудована на фрустрації та бажанні самовдосконалення. Кожна гра коштує 25 центів. Швидкий цикл перезапуску заохочує гравця грати знову і знову ("one more try"), оскільки правила прозорі, а управління елементарне.
